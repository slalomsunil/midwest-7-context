# [STORY-ID] Story Title

Story Overview
As a chat application user
I want to see all the users who are currently logged in
So that I can decide who to send my message

Story Details
Description
As a user i want to see the list of logged in users so that i can select the user and message them. This list should not inlude my own ID, and the list should be dynamic such that if the user has logged out, the name should disappear.

Business Value
One user can select another user and begin a new chat

Acceptance Criteria
Given a user has logged in, when they enter the main screen, then they must see the list of logged in users

Additional Criteria

1. if a user has logged out, the name should disappear, list gets refreshed.
2. the list should not include the logged in users id
