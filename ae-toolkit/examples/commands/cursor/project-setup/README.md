# Project Setup Commands

Cursor commands for bootstrapping AI assistance in projects.

## Commands

### initialize-ai.md
Set up Cursor configuration in a new or existing project.
- **Best Mode**: Agent Mode (creates files/directories)
- **Use**: First-time AI setup

### configure-rules.md
Create or enhance `.cursorrules` with standards from library.
- **Best Mode**: Agent Mode (creates/edits `.cursorrules`)
- **Use**: Adding or updating coding standards

## Cursor Configuration

Commands help create:
- `.cursorrules` - Project-level AI behavior and standards
- `.cursor/commands/` - Project-specific commands
- `.cursor/settings.json` - Cursor workspace settings

## Customization

Adapt for your organization by including company-specific standards, directory structures, and AI tool configurations.

See `../../generic/project-setup/` for Claude Code equivalents.
