# Git Workflow Commands

Cursor command files for common git operations and workflows.

## Commands

### commit-workflow.md
Guide through creating conventional commits.
- **Best Mode**: Agent Mode (needs terminal/git access)
- **Use**: After making code changes

### pr-creation.md
Analyze branch changes and create PR with description.
- **Best Mode**: Agent Mode (needs git/gh CLI)
- **Use**: Ready to create pull request

### branch-management.md
Create, switch, and manage branches.
- **Best Mode**: Agent Mode (needs git access)
- **Use**: Starting new work or switching branches

## Cursor Integration

Commands work with:
- **Terminal integration**: Git commands via Cursor
- **@code context**: Understand changes in project context
- **Composer**: Multi-file git operations

## Customization

Adapt for your team's git conventions by modifying command content to match your commit format, branch naming, and PR template.

See `../../generic/git-workflows/` for Claude Code equivalents.
